## 技术要点

### Better Auth Organization API
```typescript
import { authClient } from '~/lib/auth-client';

// 创建组织
await authClient.organization.create({
  name: 'My Organization',
  slug: 'my-org', // 可选
});

// 获取用户的组织列表
const { data } = await authClient.organization.list();

// 设置活动组织
await authClient.organization.setActive({
  organizationId: 'org-xxx'
});
```

## 测试指南

### 步骤 1：访问组织设置页面

1. 使用 `gaopeng@winko.ai` 登录系统
2. 点击右上角头像（用户菜单）
3. 选择 "Organization"

### 步骤 2：创建组织

1. 填写组织名称（必填）：例如 "My Organization"
2. 填写 Slug（可选）：例如 "my-org"
3. 点击 "Create Organization" 按钮
4. 看到成功提示："Organization created successfully! You are now the owner."

### 步骤 3：验证管理员权限

1. 访问权限设置页面：`/agents/settings/permissions`
2. 应该看到：
   - ✅ 当前权限模式显示
   - ✅ 角色显示为 "owner"
   - ✅ 可以编辑权限设置
   - ✅ Bash 工具开关可用（在 Bypass 模式下）

### 步骤 4：配置权限（可选）

在权限设置页面：
1. 选择权限模式：`Bypass`（高级模式）
2. 启用 Bash 工具
3. 点击 "保存设置"
